# ওমর ফারুক — পোর্টফোলিও ওয়েবসাইট

এটা একটা সিম্পল, স্ট্যাটিক HTML/CSS পোর্টফোলিও সাইট। কোনো বিল্ড টুল বা ব্যাকএন্ড লাগে না — সরাসরি GitHub Pages এ হোস্ট করা যাবে।

## ফাইলগুলো
- `index.html` — মূল পেজ
- `style.css` — ডিজাইন/স্টাইল

## GitHub Pages এ হোস্ট করার ধাপ

1. GitHub এ একটা নতুন রিপোজিটরি বানাও (যেমন: `omar-portfolio`)
2. `index.html` ও `style.css` — এই দুইটা ফাইল রিপোতে আপলোড করো
   (GitHub এর "Add file → Upload files" বাটন দিয়ে সরাসরি drag-drop করতে পারবি)
3. রিপোর **Settings → Pages** এ যাও
4. "Branch" এর জায়গায় `main` সিলেক্ট করে Save করো
5. কিছুক্ষণ পর সাইটটা লাইভ হয়ে যাবে এই লিংকে:
   `https://<তোমার-গিটহাব-ইউজারনেম>.github.io/omar-portfolio/`

## পরে যা বদলাতে পারো
- `index.html` এ `contact` সেকশনে ইমেইল আর সোশ্যাল লিংক বসাও (এখন প্লেসহোল্ডার আছে)
- `about` সেকশনের লেখা নিজের মতো করে এডিট করো
- কালার চাইলে `style.css` এর উপরের `:root` অংশে হেক্স কোড বদলে দিলেই পুরো সাইটের থিম বদলে যাবে
